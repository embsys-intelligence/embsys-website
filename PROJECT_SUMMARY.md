## 🚀 Getting Started

### 1. Install Dependencies
```bash
cd embsys_website
npm install
```

### 2. Copy Logo to Public (Important!)
```bash
cp embsys_logo.png public/embsys_logo.png
```

### 3. Run Development Server
```bash
npm run dev
```

### 4. Open in Browser
```
http://localhost:3000
```

### 5. Build for Production
```bash
npm run build
npm start
```
